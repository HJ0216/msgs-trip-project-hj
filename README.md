## About This Project ✋
마실가실 사이트는 국내 여행 일정 생성 및 여행 후기 커뮤니티 사이트 입니다.

>코로나 펜데믹으로 인하여 해외 여행길이 막히고, 
>어디론가의 여행이 필요한 여행객들의 시선이 국내로 쏟아짐에 따라 국내 여행의 
>급속도로 성장하는 계기가 되었습니다. 
>코로나 종식 이후에도  지속적인 국내 여행 활성화를 위해 각 지역의 매력적인 
>여행지 정보와 이야기를 공유할 수 있는 여행 공감 사이트를 기획하고자 하였습니다
